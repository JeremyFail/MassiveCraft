# FactionsChat

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg?style=for-the-badge)](https://opensource.org/licenses/MIT)

FactionsChat is a Minecraft Spigot/Paper server plugin that integrates with MassiveCraft Factions to allow players to communicate in chat channels.

<b>Please read the guide contained in the parent project directory if you want to compile the source of this plugin.</b>

## About
This plugin was originally created in 2020 by eirikh1996 for the original Factions3 by Madus. The code has been refactored to Java (from Kotlin) for easier maintainability with the other MassiveCraft projects that are part of this repository.

The original source can be found here: https://github.com/eirikh1996/Factions3Chat
